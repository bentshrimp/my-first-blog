저희 고딩코딩 (코딩하는 고딩들) 은 django로 이 블로그를 만들어봤습니다.



django는 python이라는 프로그래밍 언어로 만들어진 무료 오픈소스 웹 애플리케이션 프레임워크에요. 쉽고 빠르게 웹사이트를 개발할 수 있도록 돕는 구성요소로 이루어진 웹 프레임워크랍니다.

웹사이트를 구축할 때, 비슷한 유형의 요소들이 항상 필요합니다. 비슷한 것들도 새로 만들땐 처음부터 다시 만들려면 힘들겠죠? 집을 만드는데 굳이 나무를 직접 캐와야 하지는 않으니까요. 웹 프레임워크는 우리들의 이런 수고를 덜어주는 도구에요.



국가기관을 입법, 사법, 행정으로 나누듯이 웹 사이트를 비롯한 소프트웨어는 Model, View, Controller로 나눠요. 이걸 MVC 패턴이라고 부른답니다. 다만 저희가 쓰고 있는 django에선 이름만 다르게 MVT 패턴이라고 해요.

이 블로그의 게시판에서 쓰이고 읽히고 수정되는 글들은 데이터베이스의 데이터로 저장되죠. 이 데이터의 형식을 지정하고 저장하고 불러오는 작업들에 관한 코드들이 Model 파트에서 작성. 이걸 사용자가 목록과 글, 입력화면 등으로 시각적으로 볼 수 있도록 해주는 html, css 등의 요소가 Template 파트에서 작성되고요. 이 둘을 연결해주는 부분, Model의 데이터를 Template에 연결해서 사용자가 화면을 통해 데이터를 읽고 쓰고 지우고 할 수 있도록 전반적으로 제어 하는 파트가 View에요.

저희가 만든 블로그의 설명를 듣고 직접 글을 올려보세요!
